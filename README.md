# securelogin
*A fully coded backend for the login system of your website*

> On your MySql Database create a table named ***users*** with the following columns:
* **id** as an **int** with a maximum length of **11**
* **username** as a **varchar** with a maximum length of **255**
* **password** as a **varchar** with a maximum length of **255**
* **uuid** as a **varchar** with a maximum length of **255**
* **uuidset** as a **datetime**
